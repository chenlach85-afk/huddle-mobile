Build a mobile-first PWA called **Huddle** for Israeli sports team management. **Hebrew is the default language; English is available as a toggle.** The Hebrew experience must feel native, not translated — copy should sound like an Israeli coach actually wrote it.

Brand stays English: "HUDDLE". Hebrew tagline: "כל הקבוצה. במקום אחד."

One app, two roles: **Coach** (מאמן) and **Player** (שחקן). Default language Hebrew with full RTL. English toggle in profile.

## Critical Hebrew-first requirements

These rules override anything else if there's a conflict:

1. **Default language is Hebrew (`he`)** — every new user starts in Hebrew unless their browser is explicitly set to English. Stored in `profiles.preferred_language` (default `'he'`).
2. **HTML root is `<html dir="rtl" lang="he">` by default.** RTL layout is primary, not LTR-flipped.
3. **All copy is written natively in Hebrew first, then translated to English** — not the other way around. Use Israeli sports vocabulary, not literal translations.
4. **Hebrew uses Heebo font everywhere**, including display text. Don't fall back to Bebas Neue for Hebrew (no Hebrew glyphs).
5. **Date format: DD/MM/YYYY for Hebrew users**, time in 24h format. Use `date-fns/locale/he`.
6. **Day abbreviations in Hebrew use single letters**: א׳ ב׳ ג׳ ד׳ ה׳ ו׳ ש׳.
7. **Numbers stay LTR** in Hebrew text — wrap them with `dir="ltr"` when needed.
8. **Test the Hebrew experience first.** When the agent says "done," Hebrew works end-to-end before English is even verified.

**Per-team theming**: at boot, set `document.documentElement.style.setProperty('--team-color', team.primary_color)`. All accents use `var(--team-color)`. Default `#FF6B35`.

### Typography

```css
--font-display-en: 'Bebas Neue', 'Oswald', 'Impact', sans-serif;
--font-display-he: 'Heebo', sans-serif;
--font-body: 'Heebo', 'Inter', system-ui, sans-serif;
--font-numeric: 'Bebas Neue', 'Heebo', monospace;
```

**Hebrew rules**:
- Display headlines in Hebrew use Heebo weight 900
- The "HUDDLE" wordmark always uses Bebas Neue (it's a brand element)
- Body text: always Heebo
- Jersey numbers: Bebas Neue weight 900 (visual identity even in Hebrew)
- Hebrew text reads slightly larger — use 14-16px body where English would be 13-15px
- Letter-spacing: 0 or slightly negative for Hebrew, never wide tracking
- Section labels:
  - English: `text-[10px] tracking-[2px] font-bold uppercase`
  - Hebrew: `text-[11px] font-bold` (no tracking, no uppercase — Hebrew has no caps)
- Numbers in Hebrew context: wrap in `<span dir="ltr" style="unicode-bidi:embed">10</span>`

### RTL handling

- Tailwind logical properties: `ms-`, `me-`, `ps-`, `pe-`, `start-`, `end-`
- Text alignment: `text-start` / `text-end`, never `text-left` / `text-right`
- Direction-implying icons flip in RTL via:
  ```css
  [dir="rtl"] .flip-rtl { transform: scaleX(-1); }
  ```
- Timeline strips read RTL in Hebrew (newest on right)
- Swipe gestures reverse direction in RTL
- Floating "+" button stays bottom-right regardless of language (mobile thumb ergonomics)

create extension if not exists "uuid-ossp";

create table public.teams (
  id uuid primary key default uuid_generate_v4(),
  name text not null,
  sport text,
  age_group text,
  season text,
  primary_color text default '#FF6B35',
  logo_url text,
  invite_code text unique default substring(uuid_generate_v4()::text from 1 for 8),
  created_by uuid references auth.users(id),
  created_at timestamptz default now()
);

create table public.profiles (
  id uuid primary key references auth.users(id) on delete cascade,
  email text,
  full_name text,
  phone text,
  avatar_url text,
  preferred_language text default 'he' check (preferred_language in ('en', 'he')),
  push_subscription jsonb,
  created_at timestamptz default now()
);

create table public.team_members (
  id uuid primary key default uuid_generate_v4(),
  team_id uuid not null references teams(id) on delete cascade,
  user_id uuid not null references auth.users(id) on delete cascade,
  role text not null check (role in ('coach', 'assistant_coach', 'player', 'parent')),
  jersey_number int,
  position text,
  notes text,
  status text default 'active' check (status in ('active', 'inactive', 'pending')),
  joined_at timestamptz default now(),
  unique(team_id, user_id)
);

create table public.events (
  id uuid primary key default uuid_generate_v4(),
  team_id uuid not null references teams(id) on delete cascade,
  title text not null,
  type text not null check (type in ('training', 'game', 'meeting', 'tournament', 'other')),
  description text,
  starts_at timestamptz not null,
  ends_at timestamptz,
  location_name text,
  location_address text,
  location_lat numeric,
  location_lng numeric,
  arrival_time timestamptz,
  uniform_color text,
  uniform_notes text,
  what_to_bring text,
  opponent text,
  is_home boolean,
  created_by uuid references auth.users(id),
  created_at timestamptz default now(),
  updated_at timestamptz default now()
);

create index events_team_starts_idx on events(team_id, starts_at);

create table public.attendance (
  id uuid primary key default uuid_generate_v4(),
  event_id uuid not null references events(id) on delete cascade,
  user_id uuid not null references auth.users(id) on delete cascade,
  status text default 'pending' check (status in ('pending', 'confirmed', 'declined', 'maybe', 'attended', 'no_show')),
  reason text,
  responded_at timestamptz,
  unique(event_id, user_id)
);

create table public.conversations (
  id uuid primary key default uuid_generate_v4(),
  team_id uuid not null references teams(id) on delete cascade,
  type text not null check (type in ('group', 'direct')),
  title text,
  created_at timestamptz default now()
);

create table public.conversation_members (
  conversation_id uuid not null references conversations(id) on delete cascade,
  user_id uuid not null references auth.users(id) on delete cascade,
  last_read_at timestamptz default now(),
  primary key (conversation_id, user_id)
);

create table public.messages (
  id uuid primary key default uuid_generate_v4(),
  conversation_id uuid not null references conversations(id) on delete cascade,
  sender_id uuid not null references auth.users(id) on delete cascade,
  body text,
  attachment_url text,
  attachment_type text check (attachment_type in ('image', 'video', 'pdf', 'audio')),
  created_at timestamptz default now()
);

create index messages_conversation_idx on messages(conversation_id, created_at desc);

create table public.announcements (
  id uuid primary key default uuid_generate_v4(),
  team_id uuid not null references teams(id) on delete cascade,
  author_id uuid references auth.users(id),
  title text not null,
  body text,
  pinned boolean default false,
  created_at timestamptz default now()
);

create table public.announcement_reads (
  announcement_id uuid not null references announcements(id) on delete cascade,
  user_id uuid not null references auth.users(id) on delete cascade,
  read_at timestamptz default now(),

);

create table public.training_programs (
  id uuid primary key default uuid_generate_v4(),
  team_id uuid not null references teams(id) on delete cascade,
  title text not null,
  description text,
  category text,
  file_url text,
  file_type text check (file_type in ('pdf', 'video', 'image', 'link')),
  duration_minutes int,
  created_by uuid references auth.users(id),
  created_at timestamptz default now()

create table public.tasks (
  id uuid primary key default uuid_generate_v4(),
  team_id uuid not null references teams(id) on delete cascade,
  title text not null,
  description text,
  training_program_id uuid references training_programs(id) on delete set null,
  due_at timestamptz,
  requires_proof boolean default false,
  created_by uuid references auth.users(id),
  created_at timestamptz default now()

create table public.task_assignments (
  id uuid primary key default uuid_generate_v4(),
  task_id uuid not null references tasks(id) on delete cascade,
  user_id uuid not null references auth.users(id) on delete cascade,
  status text default 'assigned' check (status in ('assigned', 'in_progress', 'completed', 'skipped')),
  proof_url text,
  notes text,
  completed_at timestamptz,
  unique(task_id, user_id)

create index task_assignments_user_idx on task_assignments(user_id, status);

create table public.team_files (
  id uuid primary key default uuid_generate_v4(),
  team_id uuid not null references teams(id) on delete cascade,
  name text not null,
  url text not null,
  type text check (type in ('photo', 'video', 'pdf', 'document', 'other')),
  size_bytes bigint,
  uploaded_by uuid references auth.users(id),
  album text,
  created_at timestamptz default now()
);

create or replace function public.handle_new_user()
returns trigger language plpgsql security definer set search_path = public as $$
begin
  insert into public.profiles (id, email, full_name)
  values (new.id, new.email, coalesce(new.raw_user_meta_data->>'full_name', ''));
  return new;
end; $$;

drop trigger if exists on_auth_user_created on auth.users;
create trigger on_auth_user_created
  after insert on auth.users
  for each row execute procedure public.handle_new_user();

alter table teams enable row level security;
alter table profiles enable row level security;
alter table team_members enable row level security;
alter table events enable row level security;
alter table attendance enable row level security;
alter table conversations enable row level security;
alter table conversation_members enable row level security;
alter table messages enable row level security;
alter table announcements enable row level security;
alter table announcement_reads enable row level security;
alter table training_programs enable row level security;
alter table tasks enable row level security;
alter table task_assignments enable row level security;
alter table team_files enable row level security;

create or replace function public.is_team_member(team_uuid uuid)
returns boolean language sql security definer stable as $$
  select exists(select 1 from team_members
    where team_id = team_uuid and user_id = auth.uid() and status = 'active');
$$;

create or replace function public.is_team_coach(team_uuid uuid)
returns boolean language sql security definer stable as $$
  select exists(select 1 from team_members
    where team_id = team_uuid and user_id = auth.uid()
      and role in ('coach', 'assistant_coach') and status = 'active');
$$;

create policy "view own and teammates profiles" on profiles for select
  using (id = auth.uid() or
    exists(select 1 from team_members tm1 join team_members tm2 on tm1.team_id = tm2.team_id
           where tm1.user_id = auth.uid() and tm2.user_id = profiles.id));
create policy "update own profile" on profiles for update using (id = auth.uid());
create policy "members view team" on teams for select using (is_team_member(id));
create policy "coaches update team" on teams for update using (is_team_coach(id));
create policy "anyone create team" on teams for insert with check (created_by = auth.uid());
create policy "members view roster" on team_members for select using (is_team_member(team_id));
create policy "coaches manage roster" on team_members for all using (is_team_coach(team_id));
create policy "users join team" on team_members for insert with check (user_id = auth.uid());
create policy "members view events" on events for select using (is_team_member(team_id));
create policy "coaches manage events" on events for all using (is_team_coach(team_id));
create policy "view own attendance" on attendance for select
  using (user_id = auth.uid() or
    exists(select 1 from events e where e.id = attendance.event_id and is_team_coach(e.team_id)));
create policy "users update own attendance" on attendance for all using (user_id = auth.uid());
create policy "coaches manage attendance" on attendance for all
  using (exists(select 1 from events e where e.id = attendance.event_id and is_team_coach(e.team_id)));
create policy "members view conversations" on conversations for select
  using (exists(select 1 from conversation_members cm where cm.conversation_id = id and cm.user_id = auth.uid()));
create policy "coaches create conversations" on conversations for insert with check (is_team_coach(team_id));
create policy "members view conv members" on conversation_members for select
  using (exists(select 1 from conversation_members cm where cm.conversation_id = conversation_members.conversation_id and cm.user_id = auth.uid()));
create policy "coaches manage conv members" on conversation_members for all
  using (exists(select 1 from conversations c where c.id = conversation_id and is_team_coach(c.team_id)));
create policy "members view messages" on messages for select
  using (exists(select 1 from conversation_members cm where cm.conversation_id = messages.conversation_id and cm.user_id = auth.uid()));
create policy "members send messages" on messages for insert
  with check (sender_id = auth.uid() and
    exists(select 1 from conversation_members cm where cm.conversation_id = messages.conversation_id and cm.user_id = auth.uid()));
create policy "members view announcements" on announcements for select using (is_team_member(team_id));
create policy "coaches manage announcements" on announcements for all using (is_team_coach(team_id));
create policy "users mark read" on announcement_reads for all using (user_id = auth.uid());
create policy "members view programs" on training_programs for select using (is_team_member(team_id));
create policy "coaches manage programs" on training_programs for all using (is_team_coach(team_id));
create policy "members view team tasks" on tasks for select using (is_team_member(team_id));
create policy "coaches manage tasks" on tasks for all using (is_team_coach(team_id));
create policy "view own assignments" on task_assignments for select
  using (user_id = auth.uid() or
    exists(select 1 from tasks t where t.id = task_id and is_team_coach(t.team_id)));
create policy "users update own assignment status" on task_assignments for update using (user_id = auth.uid());
create policy "coaches manage assignments" on task_assignments for all
  using (exists(select 1 from tasks t where t.id = task_id and is_team_coach(t.team_id)));
create policy "members view files" on team_files for select using (is_team_member(team_id));
create policy "members upload files" on team_files for insert
  with check (uploaded_by = auth.uid() and is_team_member(team_id));
create policy "uploader or coach delete" on team_files for delete
  using (uploaded_by = auth.uid() or is_team_coach(team_id));

insert into storage.buckets (id, name, public) values
  ('avatars', 'avatars', true),
  ('team-logos', 'team-logos', true),
  ('team-files', 'team-files', false),
  ('message-attachments', 'message-attachments', false),
  ('task-proofs', 'task-proofs', false),
  ('training-programs', 'training-programs', false)
on conflict (id) do nothing;

create policy "authenticated upload" on storage.objects for insert to authenticated with check (true);
create policy "authenticated read" on storage.objects for select to authenticated using (true);
create policy "users delete own uploads" on storage.objects for delete to authenticated using (owner = auth.uid());

## Date and time formatting

```js
import { format } from 'date-fns';
import { he, enUS } from 'date-fns/locale';

const locale = userLanguage === 'he' ? he : enUS;

format(date, 'EEEE, d בMMMM', { locale })
// Hebrew: "שלישי, 4 במאי"
// English: "Tuesday, May 4"

format(date, 'HH:mm')  // Always 24h: "17:00"

format(date, 'dd/MM/yyyy', { locale: he })   // "04/05/2026"
format(date, 'M/d/yyyy', { locale: enUS })   // "5/4/2026"
```

For event hero subtitle "ג׳ · 17:00":
- Day letter: `format(date, 'EEEEEE', { locale: he })` → "ג׳"
- Combined: `${day} · ${time}`

3. **RTL layout is primary**, not LTR-flipped
4. **Heebo font everywhere**, including display
5. **Numbers stay LTR** in mixed text
6. **Date format DD/MM/YYYY**, time 24h
7. **Per-team color theming** via `var(--team-color)`
8. **Sport-team aesthetic**: jersey tiles, scoreboard stats, gradient hero cards
9. **Israeli vocabulary**: סגל, מגרש, מוצ"ש, etc.

- Live game scoring
- AI-generated training plans
- Statistics dashboards beyond simple attendance %
